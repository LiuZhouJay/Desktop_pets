# 桌面宠物

## 介绍

1. 可以进行远程控制，基于mqtt
2. 可以显示实习时间
3. 可以显示天气
4. 使用u8g2显示多级菜单
5. 做出一些基本的动作，比如打招呼，握手等
6. 可以进行语音控制

## 引脚配置

| 引脚 | 描述 |
| --- | --- |
| 4   | IIC SCL|
| 5   | IIC SDA|
| 0~3 | 4个舵机 |
| 18  | 按键1  |
| 19  | 按键2  |


## 注意

1. 引脚4和5为IIC总线，请勿使用。
2. 引脚0~3为4个舵机，请勿使用。
3. 引脚18和19为按键，请勿使用。
